## Curation Rules

仓库中保留的是适合公开共享的教学资料，不是原始课程目录的完整镜像。当前整理遵循以下规则：

- 保留：实验指导、作业题目、课件、课程项目发布材料、示例代码、参考论文、公开答案材料。
- 排除：软件安装镜像、供应商工具目录、综合生成目录、成绩表、评分文件、学生提交记录、课程录像、超大打包存档和纯行政留档文件。
- 对原始文件名尽量保持不变，避免破坏课程资料之间已有的引用关系。
